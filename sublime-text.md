# 1. Base Sublime Keymap.

| Command                   |             Description                                 |      Remark                           |
| ------------------------- | ------------------------------------------------------- | ------------------------------------- |
| `Ctrl` + `N`              | Open new tab.                                           |                                       |
| `Ctrl` + `Shift` + `L`    | Multiple line edit mode.                                |                                       |
| `Ctrl` + `Shift` + `P`    | Open Command Panel.                                     |                                       |
