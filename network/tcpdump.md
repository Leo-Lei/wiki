|                              命令                                               |                          说明                        |
| ------------------------------------------------------------------------------- | ---------------------------------------------------- |
| `sudo tcpdump -D`                                                               | 查看本地网络接口列表                                   |
| `sudo tcpdump -A -i lo0 '( (tcp) and (port 8088) and (host 10.66.182.103) )'`   | 监听本地localhost回环地址网卡，tcp协议，端口8088的流量  |



|           参数          |                                                   说明                                          |
| ----------------------- | ---------------------------------------------------------------------------------------------- |
| `-s 0`                  | 从一个包中截取的字节数，0表示不截断，抓完整的数据包。tcpdump默认显示部分数据，默认68字节。            |
| `-vvv`                  | 输出详细的报文信息                                                                               |
