# 域名级别

|   Level          |              Desc                                                 |
| ---------------- | ----------------------------------------------------------------- |
| 顶级域名          | `.com`,`.net`,`.cn`,gov.cn等为顶级域名                              |
| 一级域名          | `abc.com`,`example.cn`,`beijing.gov.cn`等为一级域名                 |
| 二级域名          | `www.abc.com`,`news.example.cn`,`www.beijing.gov.cn`为二级域名      |




# 域名解析
|     Type       |                           Desc                                   |
| -------------- | ---------------------------------------------------------------- |
| `A`            | 将域名指向一个IPv4地址，比如10.10.10.10                              |
| CNAME          | 将域名指向一个域名                                                  |
| MX             | 电子邮件服务                                                       |
| NS             | 添加域名解析服务器记录。如果要将子域名指定某个域名服务器来解析             |
