# if判断
|                  Command                   |                     Desc                          |
| ------------------------------------------ | ------------------------------------------------- |
| `if [ -z "${str1}" ]`                      | 如果字符串${NAME}长度为零，返回true                 |
| `if [ -n "${str1}" ]`　　　　　             | 当串的长度大于0时为真(串非空)                       |
