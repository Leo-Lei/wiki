# 快捷键

|            Key                 |                     Desc                                |
| ------------------------------ | ------------------------------------------------------- |
| `Command` + `N`                | Open Type                                               |
| `Command` + `Shift` + `N`      | Open File                                               |
| `Command` + `B`                | Declaration                                             |
| `Control` + `Option` + `H`     | Call Hierarchy                                          |


# 保存Go文件时自动格式化

GoLand --> Preferences -->  Tools --> File Watchers
添加一个`go fmt`即可。
