# java命令

|                           命令                         |                            描述                            | 
| ------------------------------------------------------ | ---------------------------------------------------------- | 
| `jar xvf hello.jar`                                    | 解压缩jar包                                                 | 
| `java -jar hello.jar`                                  | 运行jar包, 需要jar可执行。可使用maven插件或Spring Boot        |
| `java -Dfoo=bar -Dname=tom -jar hello.jar`             | 运行jar包, 并传递-D参数。注意-D参数在-jar前面。               |
