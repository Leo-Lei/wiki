# vi的3种模式：
1. Command mode.该模式下不能编辑。    
2. Insert mode.该模式下可以编辑。    
3. Last line mode.该模式下可以保存，退出等。    

# VI常用命令    

|             Command              |                          Description                              |
| -------------------------------- | ----------------------------------------------------------------- |
| 命令模式下 `dd`                    | 删除当前行                                                          |
| 命令模式下 `0`                     | 到当前行的行首                                                       |
| 命令模式下 `$`                     | 到当前行的行末                                                       |
| 命令模式下 `i`                     | 进入insert模式                                                      |
| 命令模式下 `:`                     | 进入`last line`模式                                                 |
| `last line`模式下 `wq`            | 保存并退出                                                          |
| `last line`模式下 `q!`            | 退出而不保存                                                         |
