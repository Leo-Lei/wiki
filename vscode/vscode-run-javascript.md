# VS 调试Javascript

安装一个插件，叫Run什么来着。
安装完插件后，可以执行代码文件。但是还需要安装对应语言的环境。比如，如果想直接运行Javascript文件，还需要安装Javascript环境。即安装Node。

安装完Node后，就可以在VSCode中新建一个文件，将语言设置为Javascript。可以看到标题栏的右侧有一个箭头，点击箭头就可以直接运行Javascript代码了。
